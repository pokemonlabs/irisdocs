## Available Integrations

<CardGroup cols={2}>
  <Card title="GitHub" icon="github">
    - Automatic test triggers on PRs
    - Issue creation and tracking
    - Results visualization in PR checks
    - Workflow automation
  </Card>
  <Card title="Slack" icon="slack">
    - Real-time test notifications
    - Custom alert channels
    - Test result summaries
    - Failure notifications
  </Card>
  <Card title="Discord" icon="discord">
    - Instant test status updates
    - Customizable notifications
    - Channel-specific alerts
    - Team collaboration
  </Card>
  <Card title="CI/CD Platforms" icon="gears">
    - GitHub Actions
    - CircleCI
    - Jenkins
    - GitLab CI
  </Card>
</CardGroup>

## Setting Up Integrations

<Steps>
  <Step title="Access Integration Dashboard">
    Navigate to the Integrations page in your [Iris QA Dashboard](https://app.tryiris.dev/integrations)
  </Step>
  <Step title="Choose Integration">
    Select the platform you want to integrate with
  </Step>
  <Step title="Configure Settings">
    Follow the platform-specific setup instructions
  </Step>
  <Step title="Verify Connection">
    Test the integration to ensure proper functionality
  </Step>
</Steps>

<Note>
  For detailed setup instructions for each integration, please refer to our specific integration guides or contact our support team on [Discord](https://discord.gg/uuNbJQyw6g).
</Note>
