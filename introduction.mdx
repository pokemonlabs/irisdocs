## What is Iris QA?

Iris QA is an advanced autonomous QA agent platform designed for UI testing, covering both functional and non-functional aspects of your applications. Our platform leverages cutting-edge AI models to create, manage, and execute intelligent test cases with minimal human intervention.

## Getting Started

To start using Iris QA, you'll need to:

1. Create an AI agent using your preferred model
2. Set up a project to organize your tests
3. Create test cases within your project
4. Execute test runs and analyze results

Check out our [quickstart guide](/quickstart) to begin your journey with Iris QA.
