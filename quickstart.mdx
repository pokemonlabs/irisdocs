## Configure Your First Agent

After logging in, you'll need to set up your default AI agent:

1. Go to the Agent section
2. Click on "Configure Agent"
3. Select your preferred AI provider
4. Enter the required API credentials
5. Save your configuration

<Frame>
  {/* Add Agent Configuration image here */}
  <img src="/images/agent-config.png" alt="Agent Configuration" />
</Frame>

## Create a Project

Projects help organize your tests:

1. Navigate to Projects
2. Click on "Create Project"
3. Enter a project name and description
4. Click on "Create Project" to save

<Frame>
  {/* Add Create Project image here */}
  <img src="/images/create-project.png" alt="Create Project" />
</Frame>

## Create Your First Test

With your project created, you can now add tests:

1. Click on "Create Test" within your project
2. Select the project from the dropdown
3. Enter test details (name and description)
4. Provide the target website URL
5. Describe your test scenario in natural language (e.g. Navigate to google.com and search automated QA testing using Iris)
6. Click on "Preview Interpretation" to verify
7. Click on "Create Test" to save

<Frame>
  {/* Add Create Test image here */}
  <img src="/images/create-test.png" alt="Create Test" />
</Frame>

## Run Tests

Execute your tests and view results:

1. From the project dashboard, select individual tests or click on "Run All Tests" to run all tests at once
2. The AI agent will execute the tests based on your scenarios
3. View real-time progress in the Latest Run section
4. Access detailed results in the Test Run Results section

<Frame>
  {/* Add Run Tests image here */}

## View and Share Results

Review and collaborate on test results:

1. Click on any test run to see detailed steps and outcomes
2. Share reports via email with team members
3. Access historical test runs for comparison
4. Export results as needed
