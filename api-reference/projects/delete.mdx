## Delete Project

Delete an existing project by its ID.

### Headers

<ParamField header="IRIS_ACCESS_TOKEN" type="string" required>
  Your Iris API access token
</ParamField>

### Body

<ParamField body="where.id" type="string" required>
  The unique identifier of the project to delete
</ParamField>

### Response

Returns the deleted project details upon success.

<CodeGroup>

```json Request Body
{
  "0": {
    "json": {
      "where": {
        "id": "<project-id>"
      }
    }
  }
}
```

```bash Request Example
curl -X POST 'https://app.tryiris.dev/api/trpc/project.delete' \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <your-iris-token>' \
  -d @- << EOF
{
  "0": {
    "json": {
      "where": {
        "id": "<project-id>"
      }
    }
  }
}
EOF
```

</CodeGroup>

### Response Fields

<ResponseField name="id" type="string">
  Unique identifier of the deleted project
</ResponseField>

<ResponseField name="name" type="string">
  Name of the project
</ResponseField>

  Description of the project
</ResponseField>

</ResponseField>
