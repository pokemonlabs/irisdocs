## Create Test

Create a new test case with specified parameters for automated testing.

### Headers

<ParamField header="IRIS_ACCESS_TOKEN" type="string" required>
  Your Iris API access token
</ParamField>

### Body

<ParamField body="name" type="string" required>
  Name of the test case
</ParamField>

<ParamField body="description" type="string" required>
  Description of what the test does
</ParamField>

<ParamField body="url" type="string" required>
  Target URL to test
</ParamField>

<ParamField body="naturalLanguageInput" type="string" required>
  Test steps in natural language
</ParamField>

<ParamField body="status" type="string" required>
  Initial status of the test (e.g., "DRAFT")
</ParamField>

<ParamField body="projectId" type="string" required>
  ID of the project this test belongs to
</ParamField>

<ParamField body="createdById" type="string" required>
  ID of the user creating the test
</ParamField>

### Response

Returns the created test details upon success.
