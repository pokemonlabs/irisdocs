## Update Test

Update an existing test's configuration including name, description, URL, and test instructions.

### Headers

<ParamField header="IRIS_ACCESS_TOKEN" type="string" required>
  Your Iris API access token
</ParamField>

### Body

<ParamField body="where.id" type="string" required>
  The unique identifier of the test to update
</ParamField>

<ParamField body="data.name" type="string">
  New name for the test
</ParamField>

<ParamField body="data.description" type="string">
  Updated description of the test
</ParamField>

<ParamField body="data.url" type="string">
  The URL to be tested
</ParamField>

<ParamField body="data.naturalLanguageInput" type="string">
  Test instructions in natural language
</ParamField>

<ParamField body="data.isActive" type="boolean">
  Whether the test is active
</ParamField>

### Response

Returns the updated test configuration.
