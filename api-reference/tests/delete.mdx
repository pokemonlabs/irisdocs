## Delete Test

Delete a specific test by its ID.

### Headers

<ParamField header="IRIS_ACCESS_TOKEN" type="string" required>
  Your Iris API access token
</ParamField>

### Body

<ParamField body="where" type="object" required>
  Query parameters to identify the test
  <Expandable title="Toggle object">
    <ParamField body="id" type="string" required>
      The unique identifier of the test to delete
    </ParamField>
  </Expandable>
</ParamField>

### Response

Returns the deleted test details upon success.

<CodeGroup>

```json Request Body
{
  "0": {
    "json": {
      "where": {
        "id": "<test-id>"
      }
    }
  }
}
```

```bash Request Example
curl -X POST 'https://app.tryiris.dev/api/trpc/test.delete' \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <your-iris-token>' \
  -d @- << EOF
{
  "0": {
    "json": {
      "where": {
        "id": "<test-id>"
      }
    }
  }
}
EOF
```

</CodeGroup>

### Response Fields

<ResponseField name="id" type="string">
  Unique identifier of the deleted test
</ResponseField>

<ResponseField name="name" type="string">
  Name of the test
</ResponseField>

<ResponseField name="description" type="string">
  Description of the test
</ResponseField>

<ResponseField name="url" type="string">
  URL where the test was configured to run
</ResponseField>

<ResponseField name="naturalLanguageInput" type="string">
  The test steps in natural language format
</ResponseField>

<ResponseField name="status" type="string">
  Status of the test (e.g., "DRAFT")
</ResponseField>

<ResponseField name="isActive" type="boolean">
  Whether the test was active
</ResponseField>

<ResponseField name="projectId" type="string">
  ID of the project the test belonged to
</ResponseField>

<ResponseField name="createdById" type="string">
  ID of the user who created the test
</ResponseField>

<ResponseField name="createdAt" type="date">
  Timestamp when the test was created
</ResponseField>

<ResponseField name="updatedAt" type="date">
  Timestamp when the test was last updated
</ResponseField>
