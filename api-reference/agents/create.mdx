---
title: 'Create Agent'
api: 'POST /api/v1/agents'
description: 'Create a new AI agent for UI testing'
---

## Request Body

<ParamField body="name" type="string" required>
  The name of the agent. Must be unique within your organization.
</ParamField>

<ParamField body="model" type="object" required>
  <Expandable title="properties">
    <ParamField body="provider" type="string" required>
      The AI model provider. One of: `vertex-ai`, `bedrock`, `anthropic`
    </ParamField>
    <ParamField body="version" type="string" required>
      The specific model version to use (e.g., `text-bison-32k`, `claude-2`)
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="configuration" type="object" required>
  <Expandable title="properties">
    <ParamField body="temperature" type="number">
      Model temperature (0.0 to 1.0). Default: 0.7
    </ParamField>
    <ParamField body="max_tokens" type="integer">
      Maximum tokens per response. Default: 1024
    </ParamField>
    <ParamField body="timeout_seconds" type="integer">
      Request timeout in seconds. Default: 300
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="capabilities" type="array">
  List of enabled capabilities. Available options: `ui-interaction`, `test-generation`, `self-healing`
</ParamField>

## Response

<ResponseField name="id" type="string">
  The unique identifier of the created agent
</ResponseField>

<ResponseField name="created_at" type="string">
  ISO 8601 timestamp of when the agent was created
</ResponseField>

<ResponseField name="status" type="string">
  Current status of the agent: `initializing`, `ready`, `error`
</ResponseField>

## Example Request

```bash
curl -X POST https://api.tryiris.dev/api/v1/agents \
  -H "Authorization: Bearer ${API_KEY}" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "e2e-test-agent",
    "model": {
      "provider": "vertex-ai",
      "version": "text-bison-32k"
    },
    "configuration": {
      "temperature": 0.7,
      "max_tokens": 1024,
      "timeout_seconds": 300
    },
    "capabilities": [
      "ui-interaction",
      "test-generation",
      "self-healing"
    ]
  }'
```
