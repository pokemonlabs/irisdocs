## Delete Agent

Delete an existing AI agent by its ID.

### Headers

<ParamField header="IRIS_ACCESS_TOKEN" type="string" required>
  Your Iris API access token
</ParamField>

### Body

<ParamField body="where.id" type="string" required>
  The unique identifier of the agent to delete
</ParamField>

### Response

Returns the deleted agent details upon success.

<CodeGroup>

```json Request Body
{
  "0": {
    "json": {
      "where": {
        "id": "<agent-id>"
      }
    }
  }
}
```

```bash Request Example
curl -X POST 'https://app.tryiris.dev/api/trpc/agent.delete' \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <your-iris-token>' \
  -d @- << EOF
{
  "0": {
    "json": {
      "where": {
        "id": "<agent-id>"
      }
    }
  }
}
EOF
```

</CodeGroup>

### Response Fields

<ResponseField name="id" type="string">
  Unique identifier of the deleted agent
</ResponseField>

<ResponseField name="provider" type="string">
  The AI provider that was used
</ResponseField>

<ResponseField name="modelType" type="string">
  The type of model that was used
</ResponseField>

<ResponseField name="numAgents" type="number">
  Number of agents associated
</ResponseField>

<ResponseField name="isActive" type="boolean">
  Whether the agent was active
</ResponseField>

<ResponseField name="organizationId" type="string">
  The organization ID associated with the agent
</ResponseField>

<ResponseField name="createdAt" type="date">
  Timestamp of when the agent was created
</ResponseField>

<ResponseField name="updatedAt" type="date">
  Timestamp of when the agent was last updated
